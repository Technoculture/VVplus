## Getting Started

> **Note**: This repo uses `pnpm` instead of `npm` or `yarn`

```sh
# 1. Clone this repository

# 2. Start from the root directory of the repo
cd vvplus

# 3. Install the dependencies
pnpm install

# 4. Install Husky Hooks
pnpm installhooks

# 5. 
pnpm build
```
